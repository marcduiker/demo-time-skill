# External application actions

Source: https://demotime.show/actions/external/

Use this category for: launching external apps and opening URLs in the system browser.

## Actions in this category

| Action | Purpose |
|--------|---------|
| `openPowerPoint` | Open Microsoft PowerPoint. |
| `openKeynote` | Open Apple Keynote. |

---

## `openPowerPoint`

Opens Microsoft PowerPoint directly from your demo to display or reference presentations.

**Parameters:** None documented.

**Example:**

```yaml
- action: openPowerPoint
```

**Notes / gotchas:** Microsoft PowerPoint must be installed.

---

## `openKeynote`

Opens Apple Keynote directly from your demo to showcase or reference Keynote presentations.

**Parameters:** None documented.

**Example:**

```yaml
- action: openKeynote
```

**Notes / gotchas:** Apple Keynote must be installed (macOS only).
